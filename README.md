# To-Do List

A simple, interactive to-do list application that allows users to add, remove, and manage tasks efficiently. Perfect for personal task management!



## Features

- **Add Tasks**: Easily add new tasks to your to-do list.
- **Delete Tasks**: Remove tasks from your list once completed or no longer needed.
- **Task Persistence**: Tasks remain saved even if you refresh the page (depending on implementation, can be localStorage or a database).
- **Simple UI**: Minimalist design for better user experience.
- **Responsive Design**: Works seamlessly across different screen sizes and devices.



## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
   git clone https://github.com/pushparajwastaken/to-do-list.git
   cd to-do-list
  

2. **Install dependencies:**  
   Make sure you have [Node.js](https://nodejs.org/) installed. Then, run:
   npm install


3. **Start the development server:**
   npm start
   The application should now be running at `http://localhost:3000`.

## Usage
- **Add a Task:**  
  Simply enter your task in the input field and click "Add" or press Enter to add it to the list.

- **Delete a Task:**  
  Click the delete icon next to any task you want to remove.

- **Mark Task as Completed:**  
  Check the checkbox next to the task name to mark it as completed. This can visually update the task (e.g., strike-through or change in color).

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Local Storage:** (optional) For task persistence without a server
